# **App Name**: DioVista Apartments

## Core Features:

- Multi-language Support: The entire website supports Spanish, English, Polish and Czech, using JSON files for translations and a language switcher in the header.
- Local Image Handling: All images are imported as modules from a central /lib/local-images.js file, ensuring no hardcoded string paths are used in components.
- Header Navigation: Sticky header with logo, navigation links (Interior, Services, Features, Zone, Restaurants, Sports), language switcher, theme switcher, and a 'Reservar Ahora' button, with mobile menu.
- Hero Section: Full-screen hero section with background image, title, subtitle, and a 'Reservar en Booking.com' button, linking to the booking page.
- Value Proposition Section: Text-based section summarizing the benefits and unique value of the apartment.
- Explore Area Section: Highlights nearby points of interest with images, descriptions, and links to Google Maps.
- Footer Information: Footer with logo, contact information, quick links to different sections, and copyright notice.

## Style Guidelines:

- Primary color: #3498db (RGB) - A calming blue tone reminiscent of coastal horizons.
- Background color: #f0f8ff (RGB) - A very light, desaturated blue (almost white) for a clean, airy feel.
- Accent color: #2ecc71 (RGB) - A contrasting green to highlight key actions and interactive elements.
- Headline font: 'Playfair', serif - Elegant, high-contrast, fashionable feel for headlines; body text: 'PT Sans', sans-serif - A modern and warm typeface for readability.
- Use Lucide-react icons to represent different amenities, services, and features.
- Use a grid-based layout to display services, features, restaurants, and sports activities.
- Subtle hover effects on cards and buttons to indicate interactivity.
